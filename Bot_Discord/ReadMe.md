Description : Bot discord programmé en node.js permettant à l'aide d'une commande ( exemple : !sport lenomdusport) d'afficher le podium d'un sport présent pendant les Jeux Olympiques 2024. L'affichage se retrouve sous forme d'une réponse du bot Discord sous format embedMessage dans le même channel dans lequel vous aurez posté votre demande.

Le Message reprendra les noms et prénoms des 3 meilleurs participants de la discipline ainsi que leur pays. 

Recommendation : Pour faire fonctionner le bot, il vous faudra créer le votre sur la plateforme développeurs de l'application puis l'affecter sur le serveur discord de votre choix. 
Par la suite, vous devrez ajouter un fichier .env reprenant toutes les informations de votre database ainsi que votre Token discord trouvable sur le site. 

Pour finir, il suffira d'ouvrir un terminal pour rentrer les différentes formules pour installer les dépendances nécessaires au bon fonctionnement du bot

Installation : npm install dotenv
               npm install
               npm install discord.js
               npm install mysql
               npm install nodemon
